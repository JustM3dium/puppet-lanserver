Downloading Games

  1. To download games, you need a BitTorrent Client. You can find it for x86 here or for x64 here.
  2. The client mentioned above must be installed. Click through the installation dialogs and configure your installation path if you want.
  3. Now click this URL and search for the correct game.
  4. Download the *.torrent file of the game and add it to your BitTorrent client.
  5. The download should start after a few seconds. You can find the Download in your BitTorrent download directory you configured.
  6. You should keep the client active in the background all the time, so other peers can download the file from your client too. This improves the the speed when someone else download files.

Downloading games

  1. To upload games you need a FTP client. For example Filezilla. You can find a download link for x86 here and for x64 here).
  2. Install it and click through the installation dialogs.
  3. Start Filezilla and connect to the following address: "server.lan". The port number is 22 but it's optional
  4. Now you upload *.zip, *.rar or *.7z archives into the upload directory of FTP server.
  5. When the upload is completed, there will be a *.torrent file in the upload directory of this website. Other people can download the *.torrent file and share them in the network.

Unfortunately uploading directory isn't supported. Just upload archives of a directory. If you need a program to archive a directory, you can use 7-Zip. You can download it for x86 here or for x64 here.

Chat

If you want to chat with other people, you can join the local IRC server. You can find a Web Client here. Insert a nickname and then chat happily with all other participants. There is also a desktop client for IRC called HexChat. You can download it from the /tools section and install it.
